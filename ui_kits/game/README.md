# UI Kit — BetterMonday Coffee Tycoon Game

Portrait mobile game (390×844 design width). One product. Built as an interactive clickable prototype.

**Screens:**
1. **Prologue** — sepia opening narration
2. **Character Create** — name + zodiac face + element type
3. **Main Shop** — isometric shop view with furniture card
4. **Workshop (Merge Board)** — 7×7 merge grid + order slots
5. **Collection** — slime dex + element affinity grid

Navigate via the bottom nav bar. Each screen is a JSX component in this folder.

**Note:** The game codebase contains no art yet (MVP stage with `ColorRect` placeholders). These UI recreations follow the GDD's written visual direction — warm coffee browns + cream + 단청 red accent + 5-element color system.
