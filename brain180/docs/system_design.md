# Brain180 시각화 프로그램 — 시스템 설계

## 1. 핵심 철학: 순방향 + 역방향 이중 학습

### 순방향 (텍스트 → 시각화)
```
텍스트 입력
    ↓
[LENS AI] — 저자의 4차원 뇌인지구조 추출
    ↓
D3.js 노드 그래프 시각화 (4차원 레이어별 색상 구분)
    ↓
학생이 시각화를 보며 저자의 마음 이해
```

### 역방향 (시각화 → 텍스트 재해석)
```
시각화 맵 먼저 제시
    ↓
학생이 저자처럼 생각하는 방식 내면화
    ↓
텍스트를 저자의 눈으로 다시 읽기
    ↓
4차원 해석 — "저자가 이렇게 생각했구나!"
```

## 3. 4차원 시각화 레이어

D3.js 그래프에서 4개 차원을 다음과 같이 시각화합니다:

| 차원 | 시각화 방법 | 색상 |
|------|------------|------|
| 1차원 (내용) | 노드 기본 형태 | 회색 계열 |
| 2차원 (구조) | 노드 테두리 굵기 | 초록 계열 |
| 3차원 (관계망) | 엣지 두께 + 색상 | 파랑 계열 |
| **4차원 (패턴)** | **노드 발광 효과 + 특별 마커** | **황금색** |

dimension = 4인 노드/엣지는 황금색으로 발광하며 "이것이 저자의 천재성 핵심"임을 표시합니다.


## 6. Speech 실시간 피드백 시각화

학생이 스피치할 때 언급한 노드가 즉각 활성화됩니다:

| verdict | 노드 효과 | 지속 시간 |
|---------|----------|----------|
| `correct` | 초록 글로우 + 펄스 | 1.5초 |
| `partial` | 노란 글로우 | 1초 |
| `incorrect` | 빨간 점멸 | 0.5초 |
| `off_lens` | 파란 물결 효과 | 1초 |
| `4d_achieved` | 황금 폭발 효과 | 2초 |

## 8. 8,192가지 뇌인지 행동 유형

13가지 이진 축의 조합 (2¹³ = 8,192):

| # | 축 이름 | 0 | 1 |
|---|---------|---|---|
| 1 | 처리 방향 | 귀납적 (특수→일반) | 연역적 (일반→특수) |
| 2 | 초점 이동 | 세부→전체 | 전체→세부 |
| 3 | 연결 구조 | 선형 사슬 | 네트워크 망 |
| 4 | 시간 지향 | 과거/원인 분석 | 미래/결과 설계 |
| 5 | 추상도 | 구체적 사례 | 추상적 원리 |
| 6 | 검증 방식 | 경험적/실험적 | 논리적/공리적 |
| 7 | 에너지 방향 | 수렴 (좁히기) | 발산 (넓히기) |
| 8 | 문제 프레임 | 제거형 (무엇을 없앨까) | 생성형 (무엇을 만들까) |
| 9 | 대조 전략 | 유사성 찾기 | 차이성 찾기 |
| 10 | 계층 구조 | 수평 (대등 관계) | 수직 (위계 관계) |
| 11 | 변화 수용 | 점진적 개선 | 패러다임 도약 |
| 12 | 관점 전환 | 단일 관점 유지 | 다중 관점 전환 |
| 13 | 메타 인지 | 암묵적 (무의식적) | 명시적 (의식적 모니터링) |

진로적성검사 결과 → 13자리 이진수 → 가장 잘 맞는 천재 저자 매칭

## 9. API 엔드포인트 명세

| Method | Path | 입력 | 출력 |
|--------|------|------|------|
| POST | `/api/lens` | `{text, author_info?, target_iq}` | 4차원 뇌인지구조 JSON + session_id |
| POST | `/api/coupling` | `{session_id, student_speech}` | 4차원 커플링 점수 JSON |
| POST | `/api/speech/verify` | `{session_id, sentence, context, consecutive_verdicts}` | 즉각 검증 JSON |
| POST | `/api/feedback` | `{session_id}` | 종합 피드백 + 브레인샷 처방 JSON |
| GET | `/api/session/{id}` | — | 세션 전체 데이터 |
| DELETE | `/api/session/{id}` | — | 세션 삭제 |
